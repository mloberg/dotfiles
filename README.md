# mloberg dotfiles

My dotfiles

## install

    $ git clone https://github.com/mloberg/dotfiles.git ~/.dotfiles
    $ ~/.dotfiles/script/mac

## one-line install (not recommended)

    $ bash <(curl -fsS https://raw.githubusercontent.com/mloberg/dotfiles/master/script/mac)

## updating

    $ dotfiles
