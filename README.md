# dotfiles

My dotfiles

## install

```sh
git clone https://github.com/mloberg/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## macOS bootstrap

```sh
./macos/setup.sh --init
./macos/bootstrap.sh
```
